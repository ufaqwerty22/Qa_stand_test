### Описание

Это мой пет-проект по тестированию для сайта qa-stand-login.inzhenerka.tech/login  
Затронутые виды тестирования
- Function testing
- API testing

### Запуск тестов
- Клонируете репо в вашу IDE
- Прописать в консоль - **pip install -r requirements.txt** чтоб установить зависимости
- Для запуска тестов прописать - **pytest -v ./test_api.py**

### Решеные проблемы
- *Зависимость тестов* Тесты работают изолированно друг от друга при помощи conftest.py. При каждом вызове функции создается пользователь и после выполнения функции он удаляется. Это позволяет тестам не зависеть от результатов других тестов.
